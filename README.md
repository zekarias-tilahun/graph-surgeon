# A PyTorch implementation of GraphSurgeon 

## Requirements!

-   Python 3.6+
-   PyTorch 1.6+
-   PyTorch Geometric 1.6+
-   Numpy 1.17.2+
-   Networkx 2.3+
-   SciPy 1.5.4+
-   (OPTINAL) OPTUNA 2.8.0+ ```If you wish to tune the hyper-parameters of GraphSrugeon```

## Example usage

### Training

```sh
$ python src/main.py
```

### Tuning

```sh
$ python src/tune.py
```
